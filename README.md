# Setup Instructions
### Clone the repository from GitHub: git clone https://github.com/BOJJRAJI/NxtTrendZApp-APP.git

Navigate to the project directory: cd NxtTrendZApp-APP

## **Technologies Used**
- Backend: Node.js && Express.js
- Frontend: React.js
- Database: SQLITE
  

### Install dependencies:

- Backend: cd NxtTrendZApp-Backend && npm install

- Frontend: cd NxtTrendZApp-Frontend && npm install

- Run the frontend server: npm start

- Run the backend server: npm run server
